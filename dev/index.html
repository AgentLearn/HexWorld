HexWorld.Cell — Type
Cell
  • q: column index, starting from 1
  • r: row index, starting from 1

Note:

  1. (1,1) represents the upper left corner.
  2. In Hexagons notation this structure would have been called HexagonOffsetOddQ

Units:

  • length of a hexagon side is 1
  • width is 2
  • height is sqrt(3)
